⚠️ Configure seu App ID Microsoft
Para usar OneDrive, abra o arquivo agrocall.html num editor e altere a linha:
MS_CLIENT_ID = 'SEU_CLIENT_ID'
Ver como criar o App ID ▾
1. Acesse portal.azure.com
2. Azure Active Directory → App registrations → New registration
3. Nome: AgroCall | Tipo: Accounts in any org + personal
4. Redirect URI: Single-page application (SPA) → cole a URL do seu site
5. Após criar, copie o Application (client) ID
6. Cole no arquivo onde está SEU_CLIENT_ID
